# RD Manager IA v4

Plataforma de gerenciamento de marketing para agências usando RD Station, com análises por IA.

## Secrets necessários

- `DATABASE_URL` — PostgreSQL externo (Neon.tech)
- `GROQ_API_KEY` — API do Groq
- `SECRET_KEY` — Chave JWT
- `ADMIN_PASSWORD` — Senha do admin
- `RD_CLIENT_ID` / `RD_CLIENT_SECRET` — App RD Station
- `RD_REDIRECT_URI` — URL de callback OAuth
